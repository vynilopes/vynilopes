## Sobre mim

Sou um desenvolvedor back-end em formação, com foco em **Python e Django**, construindo aplicações reais do zero enquanto pavimento meu caminho para a **Ciência de Dados**.

Minha trajetória é diferente: venho de anos de experiência em **logística e varejo** (Atacadão, Atakarejo), o que me deu visão sistêmica de negócios, disciplina operacional e sensibilidade para criar soluções que resolvem problemas concretos — não apenas técnicos.

**O que estou construindo agora:**

- 🏪 **Sistema de Gestão de Estoque com PDV** — controle de inventário em tempo real com ponto de venda integrado (Django + PostgreSQL)
- 🧠 **Plataforma Descentralizada de Conhecimento** — armazenamento, backup e compartilhamento estruturado de informações com autenticação e controle de permissões (Django + SQLite)

**Onde estou evoluindo:**

- Aprofundando back-end com Django REST Framework e APIs
- Iniciando estudos em análise de dados com Pandas e Jupyter Notebook
- Praticando SQL avançado e modelagem relacional
- Explorando automação e scripting com Python puro
